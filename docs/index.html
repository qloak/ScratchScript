Basic syntax

-
    -
  • ScratchScript's syntax is pretty similar to JavaScript syntax. If you know javascript syntax, you'll probably be able to catch on pretty quickly. Note that ScratchScript doesn't have the same keywords that JavaScript does. There are no break, let, await etc.
    • -
  • Indentation is optional
    • -
  • Functions always use parentheses
    • -
  • You can have a space after the function name before the parentheses
    • -
  • Strings are always surrounded by double quotes "
    • -
  • Numbers do not need to be surrounded by quotes
    • -
  • Everything is a function and needs parentheses, including blocks like xPos() or size()
    • -
  • The exception to the rule above are variables and lists
    • -
  • There isn't much error handling, so don't mess stuff up
    • -
  • Note: Just because a project runs in the editor preview doesn't mean it will load into Scratch if you're using invalid syntax in your code.
    • -
  • Note: To have the preview have updated code, you must recompile with the big green button. Just pressing the green flag doesn't recompile.
    • -
  • To do comments, put // at the begining of the line
    • -
    - Example script: -
    whenGreenFlagClicked()
-setVar("correct?", 0)
-repeatUntil (=($correct?, 1)) {
-    ask("What is 1 + 1?")
-    if (=(answer(), 2)) {
-        sayForSecs("You got it right!", 2)
-        setVar("correct?", 1)
-    } else {
-        sayForSecs("You got it wrong...", 2)
-    }
-}
-moveSteps(10)

Scripts

-
    -

    Scripts are sections of code begining with a hat block. For example, the whenGreenFlagClicked() block is a hat block.

    -

    To seperate scripts, use a blank line. For example:

    -
    whenGreenFlagClicked()
-moveSteps(10)
-
-whenKeyPressed("space")
-sayForSecs("You pressed space!", 2)
-
    -
-

Control Flow

-
    - For the blocks repeat, repeatUntil, while, if, and forever use curly braces {} -

    For example, to make a loop that repeats until the mouse is down, you would do:

    - -
    repeatUntil (mouseDown()) {
-    [more stuff here]
-}
    -

    To do an else for an if, just add } else {

    -
    if (condition) {
-    [code]
-} else {
-    [more code]
-}
    -
-

Variables and Lists

-
    -

    Variables and lists are created as you use them.

    -

    For example, to create and set a variable to the users input, you would do:

    -
    ask("what")
-setVar("mycoolvar", answer())
-
    -

    For a list, creating and adding a item to a list would be:

    -
    addToList("myitem", "mycoolist")
    -

    The same applies for the other list and variable functions

    -

    To reference a variable or list, you would do $varname for variables or #listname for lists

    -

    Note: "reference" in this case means when reading out the value to use in another block. For blocks that change/edit the value, use a string quoted name as an argument

ScratchScript v2.0 Language Guide

+ +

Overview

+

ScratchScript v2.0 replaces the function-heavy syntax of v1.0 with a more + natural, Scratch-like style:

    -

    You can add sprites by clicking the "New Sprite" button

    +
  • Control flow uses keyword blocks such as if ... end and + repeat until ... end.
    • +
  • Assignments use = (x = 5).
    • +
  • Operators are infix (x == y).
    • +
  • Events use when ... end blocks.
    • +
  • Loose blocks run sequentially at startup.
    • +
  • Lists use [] indexing and keywords like + add/remove.
-
-

I think that's it

- + +

Execution Model

+
    +
  1. Scripts (when … end) work like Scratch + events and run in parallel when triggered.
    2. +
  2. Loose blocks are statements outside any when + and are executed once at startup.
    3. +
  3. Variables and lists are created automatically when first + assigned.
    4. +
+ +

Syntax Rules

+ +

Comments

+
# This is a comment
+// This also works
+ +

+
when greenFlag
+    move 10 steps
+end
+
+when key "space" pressed
+    say "Space!"
+end
+
+when sprite "Cat" clicked
+    say "You clicked me!"
+end

Variables

+
score = 0
+name = "Player"
+ +

Lists

+
inventory = []
+add "sword" to inventory
+say inventory[0]
+remove "sword" from inventory

Control Flow

+
if score == 0
+    say "No points yet"
+else
+    say "Good job!"
+end
+

+    say "Waiting..."
+end

+end

Actions

+
move 10 steps
+turn 15 degrees
+go to x: 0 y: 0
+say "Hello" for 2
+ask "What is your name?"

Operators

+

Arithmetic: +, -, *, /, %
+ Comparison: ==, !=, <, >, <=, >=
+ Logic: and, or, not

+ +

Formal Grammar (EBNF)

+
Program         ::= { Script | LooseStmt }
+
+Script          ::= "when" Event StmtBlock "end"
+
+LooseStmt       ::= Stmt
+
+Event           ::= "greenFlag"
+                  | "flag"
+                  | "key" String ["pressed"]
+                  | "sprite" String "clicked"
+
+StmtBlock       ::= { Stmt }
+
+Stmt            ::= Assignment
+                  | Action
+                  | Control
+                  | Comment
+
+Assignment      ::= Identifier "=" Expr
+
+Action          ::= "move" Expr "steps"
+                  | "turn" Expr "degrees"
+                  | "go" "to" "x:" Expr "y:" Expr
+                  | "say" String ["for" Expr]
+                  | "ask" String
+                  | "add" Expr "to" Identifier
+                  | "remove" Expr "from" Identifier
+                  | "change" Identifier "by" Expr
+
+Control         ::= "if" Expr StmtBlock ["else" StmtBlock] "end"
+                  | "while" Expr StmtBlock "end"
+                  | "repeat" ["until"] Expr StmtBlock "end"
+                  | "forever" StmtBlock "end"
+
+Expr            ::= Expr BinaryOp Expr
+                  | UnaryOp Expr
+                  | Identifier
+                  | Number
+                  | String
+                  | ListAccess
+                  | "(" Expr ")"
+
+ListAccess      ::= Identifier "[" Expr "]"
+
+BinaryOp        ::= "==" | "!=" | "<" | ">" | "<=" | ">="
+                  | "+" | "-" | "*" | "/" | "%"
+                  | "and" | "or"
+
+UnaryOp         ::= "not" | "-"
+
+Identifier      ::= Letter { Letter | Digit | "_" }
+Number          ::= Digit { Digit }
+String          ::= '"' { ANY - '"' } '"'
+
+Comment         ::= "#" { ANY - LineBreak }
+                  | "//" { ANY - LineBreak }
+
+Letter          ::= "A"…"Z" | "a"…"z"
+Digit           ::= "0"…"9"
+LineBreak       ::= "\n" | "\r" | "\r\n"
+ +

Example Program

+
# Loose block, runs at startup
+say "Program started"
+
+when greenFlag
+    score = 0
+    repeat until score == 3
+        ask "What is 2 + 2?"
+        if answer == "4"
+            say "Correct!" for 2
+            score = score + 1
+        else
+            say "Wrong!" for 2
+        end
+    end
+    say "You win!"
+end
+
+when key "space" pressed
+    move 10 steps
+end
